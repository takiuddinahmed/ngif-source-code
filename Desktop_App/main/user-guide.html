USER GUIDE

Step 01

At first user will login to the NGIF system through his username and password

Step 02

After system requirements checking a dashboard will be appeared where four functional icons will available (Analyze, Staff database, View profile, User guide)

Step 03

In the analyze section user can analyze the photo with the entire database or can analyze the photo with a specific image

Step 04

User can filter from his desired category and can trigger the analyze task a bit faster

Step 05

User can select an id range from his desired category as per the preference of his searching purpose

Step 06

After analyzing user can find his desired identity from the database within a very short time and user can see the match details as well

Step 07

In the staff database section user can see the entire database of the system

Step 08

View profile section will give the user the complete view of his profile

Step 09

User can easily get the complete guideline from the user guide section

Step 10

In top right corner there is an application icon. From there user can easily view the profile, edit the user settings and password settings as well. In the application icon panel sign out function is also available so that user can easily sign out from the system
